# TECHIN 510 - Week 5: Full-Stack Transition

## Overview
Tech stack transition from Python/Streamlit to TypeScript/Next.js. REST API integration, environment variable management, and PRIMM on Next.js App Router.

## Setup
```bash
npm install
cp .env.example .env.local  # Fill in your Supabase keys (if using Level 2)
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
